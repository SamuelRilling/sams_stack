# Task: Project Foundation Setup

## Scope
Create the foundational structure for a web app that will eventually fetch data, summarize it with an LLM, and deploy to Cloudflare Pages.

## Deliverables
1. Basic HTML structure with semantic tags
2. Minimal CSS for styling
3. JavaScript file for future functionality
4. Cloudflare Pages configuration (wrangler.toml)
5. README.md with setup instructions

## Constraints
- Use vanilla HTML/CSS/JS (no frameworks)
- Keep it minimal but functional
- Follow modern web standards
- Include proper meta tags for responsiveness

## Verification Steps
1. All files should be in a logical directory structure
2. HTML should validate without errors
3. CSS should be in a separate file
4. JavaScript should be in a separate file
5. wrangler.toml should have basic Cloudflare Pages configuration

## Format
Create a single commit with all foundation files. Use clear, descriptive file names.
